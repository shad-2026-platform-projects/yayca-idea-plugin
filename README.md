# YAYCA
IDEA Plugin for visualize a.yaml job dependencies.

## Main idea
Developers, when editing a.yaml, specify the dependencies between jobs to link them and use the deps of the corresponding connected jobs.  
Over time, these dependencies become quite deep — so deep that they start affecting the execution time of the entire flow.
An YAYCA plugin parses a.yaml and displays a graph of jobs dependencies in real time.
